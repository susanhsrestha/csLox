# csLox

This repository contains my implementation of the Lox programming language in C#, inspired by the book *Crafting Interpreters*.

## 📌 Status

🚧 Work in progress — just getting started.

## 🎯 Purpose

* Learn how interpreters work
* Understand parsing and AST concepts
* Practice building systems-level logic in C#

## 🛠️ Tech Stack

* C#
* .NET

## ▶️ Running the Project

```bash
dotnet run
```
